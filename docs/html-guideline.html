adjustment-lab｜HTML構成ガイドライン

全教材・製品ページを統一設計で構築するための基本ルール

1. ディレクトリ構造

/products/
│
├── index.html          ← LITE＋STANDARD 全シリーズ一覧
│
├── lite/               ← LITE教材（買い切り）
│    ├── index.html     ← A〜E ハブ（カウンセリング直結）
│    ├── A.html
│    ├── B.html
│    ├── C.html
│    ├── D.html
│    └── E.html
│
└── standard/           ← STANDARD教材（記録・分析付き）
     ├── index.html
     └── A.html ... etc

2. 共通設定

全ページに以下の設定を共通使用。

<link rel="manifest" href="../manifest.json">
<meta name="theme-color" content="#f8faf9">

:root {
  --bg: #f8faf9;
  --text: #1a1a1a;
  --accent: #00b386;
  --sub: #666;
  font-family: 'Noto Sans JP', sans-serif;
}

フッターは共通ファイルを読み込み：

<link rel="stylesheet" href="/adjustment-lab/assets/footer-lite.css">
<div id="footer"></div>
<script>
fetch("/adjustment-lab/assets/footer-lite.html")
  .then(res => res.text())
  .then(data => document.getElementById("footer").innerHTML = data);
</script>

3. ページ構成テンプレート

  • <section class="hero">：タイトル・サブテキスト
  • <section>：理解パート・ドリルなど本文
  • <section class="actions">：購入導線・戻るボタン
  • <footer>：共通読込

4. 導線ルール

FromTo内容
/products/index.html/products/lite/index.html▶ アスリート向けLITEを見る
/products/lite/index.html/products/index.html← LITEシリーズ全体へ戻る
/products/lite/index.html/products/lite/A.html各教材「詳細を見る」ボタン
/products/lite/A.html/products/counseling/index.html「カウンセリングに戻る」ボタン
/products/lite/A.html外部ストアLITE購入・STANDARDアップグレード

5. HTML生成時の指定文

新規ページ生成時には、必ず以下を依頼文の冒頭に記載する：

「既存構成（/products配下）を維持して、
既存のmanifest・footer・色設定を共通利用する形で生成してください。」

6. 補足メモ

  • LITEシリーズ： 単体買い切り教材。A〜E、01〜05。
  • STANDARDシリーズ： LITEの発展版（全10種目＋記録機能）。
  • manifest.json： すべてのPWA設定を統合。
  • footer-lite.html： ブランド統一のフッター。
  • 相対パス： ルート相対（/adjustment-lab/...）で統一。
